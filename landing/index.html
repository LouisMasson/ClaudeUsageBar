configuration

Three steps. Five minutes.

On first launch a config window opens. Here's what you need.

01

Organization ID required

Go to claude.ai/settings/usage, open DevTools (Cmd+Option+I), Network tab → filter XHR, reload. Look for a request to /api/organizations/.../usage. Copy the UUID from the URL.
# URL looks like:
/api/organizations/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/usage
02

Session Cookie required

In the same request, open the Headers tab → Request Headers → Cookie. Copy only the sessionKey=sk-ant-sid01-… part. Stored securely in macOS Keychain — never leaves your machine.
# Copy exactly this part from the Cookie header:
sessionKey=sk-ant-sid01-...
03

OpenRouter API Key optional

If you use OpenRouter, create a read-only API key and paste it into Settings. The app calls GET /api/v1/credits only — it can't touch your balance or generate requests. Leave it empty to hide the OpenRouter section entirely.
⚠ Session cookies expire after ~30 days. If the bar stops updating, grab a fresh sessionKey from DevTools and paste it into Settings (gear icon).
